# Card29 Offline - GitHub Actions Builder

This repository contains an Android project (Kotlin) for an **offline single-player 6-player Card-29 game**
(1 human + 5 bots). The app includes three bot difficulty levels: Easy, Medium, Hard.

**How to use**
1. Push this project to your GitHub repository (e.g., `card29-offline-game`).
2. The included GitHub Actions workflow will automatically build the debug APK.
3. After the workflow finishes, download the APK from the Actions artifacts.

**Notes**
- No coding required on your side.
- The workflow builds the `app-debug.apk` and uploads it as an artifact.
